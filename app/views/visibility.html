Visibility classes are merely a way to show/hide content based on foundation supplied classes (which have media queries "behind the scenes"). Similarly to breakpoints, the established class names contain small, medium, large, xlarge, and xxlarge. Specifically, the class names are:

Size Show Class Hide Class
{{ size.size }} {{ size.show }} {{ size.hide }}

In addition, the classes (both show and hide) can be modified with the up suffix. For example, if you want to only show a particular element from medium breakpoints on up, the class name would be show-for-medium-up.
